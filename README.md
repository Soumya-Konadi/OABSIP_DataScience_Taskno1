# Iris Flower Classification using Machine Learning

## Project Overview
This project focuses on the classification of Iris flowers using Machine Learning techniques in Python. The famous Iris dataset is used to analyze flower measurements and classify flowers into three species:

- Setosa
- Versicolor
- Virginica

The project includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Feature scaling
- Machine learning model preparation
- Correlation analysis

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Import Libraries
2. Load Dataset
3. Create DataFrame
4. Data Preprocessing
5. Exploratory Data Analysis
6. Feature Selection
7. Feature Scaling

## How to Run
1. Install required libraries
2. Open Jupyter Notebook
3. Run Iris.ipynb

## Conclusion
This project demonstrates the complete workflow of a Machine Learning classification project using the Iris dataset.
